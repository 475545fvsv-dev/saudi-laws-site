# 🏛️ موقع البحث الذكي في الأنظمة السعودية

## هيكل المشروع

```
saudi-laws-site/
├── index.html        ← الواجهة الأمامية (Frontend)
├── vercel.json       ← إعدادات Vercel
├── api/
│   └── search.js     ← الخادم الخلفي الآمن (Backend)
└── README.md
```

---

## 🚀 خطوات النشر على Vercel (مجاناً)

### الخطوة 1 — إنشاء حساب Anthropic والحصول على المفتاح

1. اذهب إلى https://console.anthropic.com
2. أنشئ حساباً جديداً
3. من القائمة اليسرى اختر **API Keys**
4. اضغط **+ Create Key**
5. انسخ المفتاح (يبدأ بـ `sk-ant-api...`)

---

### الخطوة 2 — رفع المشروع على GitHub

1. اذهب إلى https://github.com وأنشئ حساباً إن لم يكن لديك
2. اضغط **New Repository** واسمِّه `saudi-laws-site`
3. ارفع ملفات المشروع الثلاثة:
   - `index.html`
   - `vercel.json`
   - `api/search.js`

---

### الخطوة 3 — النشر على Vercel

1. اذهب إلى https://vercel.com وسجّل الدخول بحساب GitHub
2. اضغط **Add New Project**
3. اختر مستودع `saudi-laws-site`
4. اضغط **Deploy** (لا تغيّر أي إعدادات)

---

### الخطوة 4 — إضافة مفتاح API بشكل آمن ✅

هذه الخطوة الأهم — المفتاح لن يظهر في الكود أبداً:

1. في لوحة تحكم Vercel، اختر مشروعك
2. اذهب إلى **Settings** ← **Environment Variables**
3. اضغط **Add New**:
   - **Name:** `ANTHROPIC_API_KEY`
   - **Value:** `sk-ant-api...` (المفتاح الذي نسخته)
4. اضغط **Save**
5. اذهب إلى **Deployments** واضغط **Redeploy**

---

## ✅ بعد النشر

سيعطيك Vercel رابطاً مثل:
```
https://saudi-laws-site.vercel.app
```

هذا الرابط جاهز للمشاركة والاستخدام!

## 🔒 لماذا هذه الطريقة آمنة؟

| المشكلة القديمة | الحل الجديد |
|---|---|
| المفتاح ظاهر في HTML | المفتاح في متغيرات البيئة فقط |
| أي شخص يسرق المفتاح | لا أحد يرى المفتاح |
| فاتورة غير متوقعة | أنت تتحكم بمن يستخدم الموقع |

---

## 💡 تخصيص الموقع

لتعديل الأنظمة في قاعدة المعرفة، افتح `api/search.js` وعدّل المتغير `LAWS_SYSTEM_PROMPT`.
